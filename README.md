# Lung Cancer Detection

This project is focused on detecting lung cancer using machine learning and data analysis workflows, including notebooks and a detailed project report.

---

##  Repository Contents

| File | Description |
|------|-------------|
| `Lung cancer detection report.pdf` | Detailed project documentation (methodology, results, discussion) |
| `Lung_Cancer_B3.ipynb` | Main Jupyter notebook: data processing, modeling, results generation |
| `Lung_cancer_metrics_B3.ipynb` | Notebook dedicated to computing and visualizing performance metrics |
| `results_B3_lung.csv` | Tabulated results: model evaluation metrics for quick reference |
